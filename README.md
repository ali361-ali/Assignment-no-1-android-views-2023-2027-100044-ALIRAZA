Assignment No. 1 – Mobile App. Development



Project Overview:



This Android application is developed as part of Assignment No. 1 for the Mobile Application Development course. The purpose of this project is to demonstrate practical understanding and implementation of core Android UI Views using Java in Android Studio.



The application contains a main menu screen that allows navigation to separate screens, where each screen demonstrates a specific Android View with basic functionality.



Objectives:



Practice and understand commonly used Android UI Views



Implement Views using Java and XML layouts



Create a clean, functional, and beginner-friendly Android project



Demonstrate View navigation using Intents



Submit a complete Android Studio project for academic evaluation



Features:



One MainActivity acting as a navigation menu



Separate Activity for each Android View



Proper UI spacing using margins and padding



Simple and functional demonstrations (not static UI)



Clean and readable code structure



Beginner-friendly implementation
